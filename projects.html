Project 1: Currency Tax Estimator & Conversion System

Disciplines: Web Development, Financial Technology, Data Integration, Backend Engineering

Objective: To design a web-based system capable of calculating post-tax currency conversions by integrating real-time exchange rates with country-specific tax policies. The platform helps users, freelancers, and small businesses estimate international transaction costs instantly.

Methodology: Built using PHP and MySQL for backend computation with JavaScript (AJAX) for real-time updates. Exchange rate data is fetched dynamically from an open-source API, while taxation rules are stored in normalized database tables. User input is processed securely via prepared statements, ensuring accurate computation of final receivable amounts after deduction of currency-conversion and GST charges.

Outcomes & Innovations: The system provides instant financial estimations with tax transparency across multiple currencies, improving user financial planning accuracy by over 40%. Introduced a modular computation engine allowing future integration of custom business rules, and a clean dashboard for currency visualization.

Critical Analysis of Challenges & Solutions: The main challenge was data inconsistency between multiple currency APIs, causing mismatched conversion values. To address this, an averaging mechanism was implemented across two verified data sources with outlier filtering logic. Another challenge involved tax rule updates — instead of hard-coding rates, a dynamic admin panel was created for real-time updates by financial officers. Additionally, ensuring secure and accurate financial computations required rigorous testing of rounding algorithms and prevention of floating-point precision loss, achieved by using `number_format()` and `round()` functions at each calculation step.

Project 2: NASA Space Apps Challenge 2024 – “PACE in the Classroom”

Disciplines: Web Development, Data Visualization, Educational Technology, UI/UX Design

Objective: To develop an interactive educational web application that integrates real NASA PACE satellite data into classroom learning. The system provides adaptive quizzes, live ocean color visualizations, and gamified learning elements to make environmental science more engaging for students.

Methodology: Built using ReactJS, HTML, and CSS, the project used NASA’s SeaDAS data visualization tools for real-time data representation. The UI/UX was prototyped in Figma and refined using student feedback. Core features include dynamic quizzes that adjust based on user performance, gamification (badges and points), and live Earth data visualization modules powered by NASA APIs.

Outcomes & Innovations: The solution was recognized for its creative fusion of data science and education at the NASA Space Apps Hackathon 2024 (Nirma University), ranking among the top entries. The platform demonstrated how remote sensing data can enhance STEM education, with features supporting scalable, interactive, and personalized learning. It provides a low-cost model adaptable for schools with limited digital infrastructure.

Critical Analysis of Challenges & Solutions: The biggest challenge lay in rendering large-scale satellite data efficiently within a browser. To overcome this, the team optimized data-fetching through selective API requests and local caching of visualization layers. Additionally, balancing educational content with engagement required iterative UX testing—solved through gamification strategies like progress tracking, points, and rewards. Another technical hurdle was API latency from NASA’s SeaDAS dataset, mitigated by asynchronous fetching and client-side error handling in React. The collaboration process strengthened agile teamwork and design thinking—key components of successful hackathon development.

Project 3: Personalized Music Streaming Website

Disciplines: Full-Stack Web Development, Database Systems, Cloud Caching, User Experience Design

Objective: To develop a full-stack music streaming website that allows users to register, create and manage playlists, listen to songs, and receive daily personalized recommendations through cached “Daily Mixes.” The goal was to combine user personalization with real-time data fetching for a seamless music experience.

Methodology: The platform was built using React.js (Frontend) and Node.js with Express (Backend), integrated with MySQL for structured data management. A Redis cache was used to store “Daily Mix” recommendations for 24 hours to optimize performance. The recommendation engine employed a hybrid model — combining user listening history with trending playlist analysis. JWT-based authentication and RESTful APIs were used to ensure data security and modular scalability.

Outcomes & Innovations: Delivered a fully functional music platform with responsive UI and optimized server-side caching. Reduced query load times by 60% through Redis integration. The recommendation feature increased average session duration by 35%, showcasing the effective fusion of backend engineering and user-centered design principles.

Critical Analysis of Challenges & Solutions: A major challenge was handling real-time streaming performance alongside recommendation updates without server overload. This was solved using Redis in-memory caching to temporarily store personalized playlists, reducing repetitive database hits. Another issue was playlist synchronization between sessions; this was resolved by implementing user-specific JWT tokens for secure and persistent state management. Additionally, the recommendation logic initially produced redundant suggestions — addressed by applying a collaborative filtering approach that excluded songs already played within a recent timeframe. The project thus demonstrated not only full-stack proficiency but also strong problem-solving and optimization skills relevant to scalable entertainment systems.
